Custom Styling Example

This example shows how you can customize the appearance of the outline toggle button and outline colors.

Customizations Applied

  • Position: Moved to top-left corner
  • Styling: Gradient backgrounds with custom colors
  • Animation: Enhanced hover effects
  • Outline Colors: Custom colors optimized for dark theme

CSS Overrides Used

Here are the key CSS overrides used in this example:

Button Position

.outline-toggle-btn {
    right: auto !important;
    left: 20px !important;
    top: 20px !important;
    bottom: auto !important;
}

Custom State Colors

.outline-toggle-btn.state-enhanced {
    background: linear-gradient(45deg, #4facfe, #00f2fe) !important;
}

Dark Theme Outline Colors

.outline-mode-enhanced h1, .outline-mode-enhanced h2, /* ... */ {
    outline-color: #2ed573 !important;
}
